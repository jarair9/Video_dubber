# Supported Language Codes
Use these codes in `src/config.py` for `TARGET_LANGUAGE`.

> [!WARNING]
> The TTS Engine (Chatterbox) officialy supports the following 13-16 languages.
> Other languages (like Urdu) *might* work by falling back to a similar language (e.g. Hindi) or English phonemes, but results may vary.

| Language   | Code | Status |
| :--------- | :--- | :--- |
| **English**| `en` | ✅ Supported |
| **Hindi**  | `hi` | ✅ Supported |
| **Japanese**| `ja` | ✅ Supported |
| **Chinese**| `zh` | ✅ Supported |
| **Korean** | `ko` | ✅ Supported |
| French     | `fr` | ✅ Supported |
| German     | `de` | ✅ Supported |
| Spanish    | `es` | ✅ Supported |
| Russian    | `ru` | ✅ Supported |
| Portuguese | `pt` | ✅ Supported |
| Polish     | `pl` | ✅ Supported |
| Turkish    | `tr` | ✅ Supported |
| Italian    | `it` | ✅ Supported |
| Dutch      | `nl` | ✅ Supported |
| Czech      | `cs` | ✅ Supported |
| Arabic     | `ar` | ✅ Supported |
| **Urdu**   | `ur` | ⚠️ Experimental (Try `hi` if fails) |
| Bengali    | `bn` | ⚠️ Experimental |
| Punjabi    | `pa` | ⚠️ Experimental |
